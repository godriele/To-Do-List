## To enter your virtual environment

```bash
# This will create you flask-auth folder with /bin , /include, and /lin
python3 -m venv flask-auth


# This will activate your venv
source flask-auth/bin/activate


# To stop it just type
deactivate
```

### - Install these things inside your Virtual Environment, even if you have it installed globally
pip install flask Flask_SQLAlchemy
